# Post ideas

Format: one idea per line, starting with `- [ ]`.

- `[ ]` = not used yet (eligible for generation)
- `[x]` = already posted
- `[~]` = skipped, don't pick again

The bot only picks `[ ]` items. It marks them automatically on approve/skip.

Add new ideas any time — just append a new `- [ ]` line and commit. The next cron run will have them available.
